# Scrape Coretax Faktur Keluaran

![Layout](./image.png "Layout")

Untuk yang mengalami timeout saat export faktur keluaran ke excel, extensi ini membantu untuk melakukan export dengan lebih efisian dan lebih cepat dibanding export bawaan.

## Apa yang disupport

Saat ini hanya support export faktur keluaran.
